### Prerequisites

This project use Python 3 and needs following packages.

```
matplotlib
numpy
pandas
```

### Installing

Clone this project and install by requirements.txt.


```
git clone https://github.com/ysam12345/nn_hw2
```

And run pip install

```
cd nn_hw1
pip install -r requirement.txt
```


## Running
Just run the tk_main.py
```
python tk_main.py
```
or
```
python3 tk_main.py
```
